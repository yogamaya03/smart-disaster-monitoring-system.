# Smart Disaster Monitoring System

## Overview
This project is an IoT-based Smart Disaster Monitoring System developed using Wokwi. It detects disaster conditions and provides real-time alerts.

## Features
- Real-time monitoring
- IoT-based solution
- Emergency alert system
